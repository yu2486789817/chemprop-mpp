# 本次更新内容：加入 Lipophilicity 双数据集实验

## 1. 更新目标

原项目主要围绕 ESOL 水溶解度数据集展开，能够展示 Chemprop MPNN 在小规模分子性质预测任务上的效果。为了让实验更完整、更有 AI for Science 的研究深度，本次新增 MoleculeNet Lipophilicity 数据集，将项目扩展为双数据集分子性质预测实验。

新增后的项目同时包含两类 ADMET 相关任务：

- ESOL：预测水溶解度 LogS，关注分子在水中的溶解能力。
- Lipophilicity：预测脂溶性 logD，关注分子在脂相/水相之间的分配行为。

这两个任务形成了很自然的对照：水溶性和脂溶性都是药物发现中的关键性质，但反映的是不同的分子行为。

## 3. Lipophilicity 数据集实验介绍

### 3.1 数据集背景

Lipophilicity 是 MoleculeNet 中常用的分子性质预测数据集，预测目标通常是实验测得的 logD。logD 描述分子在脂相和水相之间的分配倾向，和药物的膜通透性、吸收、分布以及溶解性风险密切相关。

相比 ESOL，Lipophilicity 的特点是：

- 数据规模更大，约 4,200 个分子。
- 预测目标从水溶解度 LogS 换成脂溶性 logD。
- 更贴近药物发现中常见的 ADMET 性质筛选。
- 仍然可以在个人电脑上完成训练和实验。

本项目使用的数据来源为 DeepChem 公开的 MoleculeNet Lipophilicity CSV：

```text
https://deepchemdata.s3-us-west-1.amazonaws.com/datasets/Lipophilicity.csv
```

原始数据中的主要列为：

- `smiles`
- `exp`

其中 `exp` 被统一转换为项目中的 `target` 列。

### 3.2 训练设置

Lipophilicity 实验沿用了 ESOL 的模型结构，便于比较不同数据集上的表现。

模型设置为：

- Chemprop MPNN
- hidden size = 300
- message passing depth = 5
- dropout = 0.1
- batch size = 64
- early stopping patience = 15
- 指标：RMSE、MAE、R²

训练/测试划分为：

- 训练集：3,360 个分子
- 测试集：840 个分子

### 3.3 Baseline 实验结果

Lipophilicity baseline 结果保存在 `models/results_lipo_train.json`。

当前结果为：

- 训练集分子数：3,360
- 测试集分子数：840
- 实际训练轮数：86
- 最佳 epoch：71
- 训练耗时：约 347.9 秒
- RMSE：0.542
- MAE：0.399
- R²：0.801

这个结果说明，Chemprop MPNN 能够在 Lipophilicity 数据集上学到比较稳定的结构-性质关系。R² 达到约 0.80，说明模型可以解释测试集 logD 变化中的大部分方差；RMSE 约 0.54，说明预测误差处于一个相对可用的范围。

与 ESOL baseline 相比：

- ESOL baseline：RMSE=0.584，MAE=0.427，R²=0.928
- Lipophilicity baseline：RMSE=0.542，MAE=0.399，R²=0.801

需要注意，两个数据集的目标性质不同，不能只用 RMSE 数值直接判断哪个任务更简单。ESOL 的 R² 更高，说明模型在 ESOL 测试集上的解释能力更强；Lipophilicity 虽然 RMSE 略低，但 R² 低于 ESOL，说明该任务的可解释方差比例更低，可能受到分子离子化状态、实验条件和 logD 本身复杂性的影响。

### 3.4 数据规模实验结果

Lipophilicity 数据规模实验结果保存在 `results/data_scale_results_lipo.json`。

不同训练规模下的汇总结果如下：

- 20% 训练数据：672 个训练分子，RMSE=0.731，R²=0.638
- 50% 训练数据：1,680 个训练分子，RMSE=0.619，R²=0.741
- 80% 训练数据：2,688 个训练分子，RMSE=0.574，R²=0.777
- 100% 训练数据：3,360 个训练分子，RMSE=0.543，R²=0.801

可以看到，随着训练数据增加，Lipophilicity 的 RMSE 持续下降，R² 持续上升。

从 20% 到 100%：

- RMSE 从 0.731 降到 0.543。
- R² 从 0.638 提高到 0.801。

这说明更多训练数据确实提升了模型对 logD 的预测能力。与 ESOL 相比，Lipophilicity 的数据规模更大，3 次重复实验之间的 RMSE 标准差也比较小，例如 100% 数据下 RMSE 标准差约为 0.0027，说明训练结果较稳定。

ESOL 的数据规模实验也呈现类似趋势：

- 20% 训练数据：RMSE=0.958，R²=0.805
- 50% 训练数据：RMSE=0.732，R²=0.887
- 80% 训练数据：RMSE=0.614，R²=0.920
- 100% 训练数据：RMSE=0.585，R²=0.928

这进一步支持了本项目的核心结论：在分子性质预测任务中，数据量会显著影响模型性能。

### 3.6 分子案例结果

Lipophilicity 分子案例结果保存在 `results/lipo_molecule_cases.json`。

部分预测结果如下：

- Glucose：predicted logD = -2.766，低脂溶性。
- Vitamin C：predicted logD = -1.035，低脂溶性。
- Caffeine：predicted logD = -0.271，低脂溶性。
- Benzene：predicted logD = 2.706，中等脂溶性。
- Testosterone：predicted logD = 3.206，高脂溶性。
- Tamoxifen：predicted logD = 3.649，高脂溶性。

这些案例整体上符合直觉：含有较多羟基、羰基或杂原子的分子，例如 Glucose 和 Vitamin C，预测为低脂溶性；结构更疏水、芳香环或甾体骨架更明显的分子，例如 Benzene、Testosterone 和 Tamoxifen，预测为中等或高脂溶性。

需要注意，RDKit 计算的 LogP 和模型预测的 logD 不是完全相同的概念。LogP 通常描述中性分子的辛醇/水分配，而 logD 会受到分子在特定 pH 下的电离状态影响。因此某些带有酸碱官能团的分子，预测 logD 可能和简单的 RDKit LogP 有差异，这是合理的。

## 4. 对新增实验的整体分析

加入 Lipophilicity 后，项目从单一 ESOL 水溶解度预测，扩展为两个互补的分子 ADMET 性质预测任务。

这带来了几个提升：

1. 实验内容更完整。

   只做 ESOL 时，项目更像一个入门级单数据集示例。加入 Lipophilicity 后，可以说明同一套 Chemprop MPNN 流程能迁移到另一个分子性质任务上。

2. 数据规模更有层次。

   ESOL 约 1,128 个分子，Lipophilicity 约 4,200 个分子。两个数据集规模不同，便于展示小数据集和中等数据集下模型表现的差异。

3. 科学问题更丰富。

   ESOL 关注水溶性，Lipophilicity 关注脂溶性。这两个性质都和药物发现密切相关，而且经常存在 trade-off：过低脂溶性可能影响膜通透性，过高脂溶性又可能带来溶解度和非特异性结合问题。

4. 结果更适合报告展示。

   当前 `results/` 文件夹中已经包含 ESOL 和 Lipophilicity 两套预测图、误差图、数据规模图、训练曲线和分子案例图，可以支撑更完整的实验报告或 PPT。

## 5. 目前可以得出的结论

从当前结果看，Chemprop MPNN 在两个数据集上都取得了较好的回归效果。

ESOL 上，模型达到 RMSE=0.584、R²=0.928，说明对水溶解度 LogS 的预测效果较强。

Lipophilicity 上，模型达到 RMSE=0.542、R²=0.801，说明对脂溶性 logD 也具有较好的预测能力。

数据规模实验显示，两个任务都受益于更多训练数据。Lipophilicity 从 20% 到 100% 训练数据时，RMSE 从 0.731 降到 0.543，R² 从 0.638 提升到 0.801，趋势非常清晰。

因此，本项目现在不仅可以展示 Chemprop 在 ESOL 上的经典分子性质预测流程，也可以进一步展示模型在更大规模 Lipophilicity 数据集上的泛化能力，使研究内容更充实，也更符合 AI for Science 的实验叙事。
